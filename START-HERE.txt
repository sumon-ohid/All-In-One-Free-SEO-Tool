The fastest way to use this app:

  Windows:        Double-click  seo.cmd
  Mac / Linux:    Open Terminal here, run  ./seo.sh
  Docker:        docker compose up -d

The app opens in your browser at http://localhost:3000
(or whichever port the installer picked — check the Desktop guide)

------------------------------------------------------------
KEY FILES IN THIS FOLDER
------------------------------------------------------------

  seo.cmd / seo.sh       <- LAUNCHER — start/stop the server
  data.db                <- Your SQLite database
                            (clients, keywords, audits, reports —
                            BACK THIS UP regularly)
  .seo-encryption-key    <- Decrypts your stored API keys
                            (BACK THIS UP with data.db)
  .env.local             <- Env config (set APP_PASSWORD here)
  dev-server.log         <- Runtime log — tail this when debugging
  .dev-server.pid        <- PID of the running server

  README.md              <- Full feature list + install + license
  TROUBLESHOOTING.md     <- 12-section support doc (most issues)
  ROADMAP.md             <- What's coming next
  docs/HOSTING.md        <- Production hosting (Hetzner, Railway, etc)
  docs/screenshots/      <- App screenshots
  wordpress-plugin/      <- Companion WordPress plugin (.zip and install)

------------------------------------------------------------
COMMON COMMANDS
------------------------------------------------------------

Start the server          Double-click seo.cmd (Windows) / ./seo.sh (Mac/Linux)
Stop the server           In app: profile menu -> System health -> Shutdown
                          Or: kill the process whose PID is in .dev-server.pid

Update to latest          Re-run the installer one-liner (see README.md)
Backup                    In app: Settings -> Backup & restore -> Download
Restore                   Same screen: upload .db file
Forgot APP_PASSWORD       Edit .env.local, remove the APP_PASSWORD line, restart

View live logs            tail -f dev-server.log
Connect WordPress         Install wordpress-plugin/, paste connection key in app

------------------------------------------------------------
FIRST 5 MINUTES (RECOMMENDED)
------------------------------------------------------------

1. Open http://localhost:3000 (or your port)
2. Add a client at /clients/new — paste any domain
   (tool auto-detects niche + tech stack)
3. Pick an AI provider at /settings -> AI:
     - Ollama (free, local, fully offline)
     - Gemini / Groq / OpenRouter / DeepSeek (free tiers)
     - OpenAI / Anthropic / Perplexity (paid, BYO key)
4. Connect Google at /settings -> Integrations (free GSC + GA4 + PSI)
5. Run your first audit

Tomorrow: the daily AI agent kicks in automatically — runs ~17
automated jobs per client (rank checks, audit deltas, content
decay, backlink scans, GBP monitoring, alerts).

------------------------------------------------------------
PRIVACY & DATA OWNERSHIP
------------------------------------------------------------

* Your data NEVER leaves this machine
* No telemetry, no phone-home, no analytics
* Single SQLite file (data.db) holds everything
* API keys + OAuth tokens encrypted at rest (AES-256-GCM)
* The only outbound calls are:
    - Google's free APIs (only when you connect them)
    - Your chosen AI provider (only when you use AI features)
    - SERP scraping (only when you check rankings)
